# 武印视界.com 官网模块架构设计方案 (V1)

## 1. 项目概览
**定位**：武印阁生态的“数字中枢”与“全球门户”，集文化叙事、生态展示、用户互动、商业转化于一体。
**视觉风格**：玄黑 (#050505) + 朱红 (#E61C23) 色调，融合东方武道哲学与现代赛博视效。
**核心交互**：电影级视频画中画、六边形模块枢纽、3D 视差转场。

## 2. 技术栈
- **框架**：Vite + React + TypeScript
- **样式**：Tailwind CSS v4 (Vanilla CSS fallback for custom effects)
- **动画**：Framer Motion + CSS Keyframes
- **图标/矢量**：Lucide React + Custom SVG
- **路由**：React Router DOM (6+)

## 3. 模块详细设计

### 模块一：首页 (引力核) - Landing Page
- **Hero 区**：全屏自适应视频背景，文字“武印视界”动态渐现。
- **武印宣言**：视差滚动效果，背景为武者练功剪影。
- **三大价值入口**：文化、科技、金融，卡片悬浮展开。
- **六大生态矩阵**：六边形蜂窝布局交互，点亮红光特效。
- **最新动态**：时间轴/电影胶片式横向滚动。

### 模块二：武印宇宙 (文化叙事) - Lore & Culture
- **Universe Map**：视频热点交互。背景为飞越地图的 CG 视频，浮现可点击点位。
- **哲学根基**：“止·戈·印”三段式排版，结合水墨动态笔触。
- **人物谱系**：宗师/武者卡牌墙，支持点击翻转查看属性。
- **非遗融合**：图文 + 视频片段，展示龙泉剑、丝绸等元素。

### 模块三：武印视界 (赛事盛典) - Event Showcase
- **2026 杭州首秀**：赛事信息流，购票入口设计为“通关文牒”样式。
- **仪式感剧场**：天罡三十六阶流程化分步展示。
- **武者阵容**：双重视觉对决布局（传统 vs 机甲）。
- **往期回顾**：Bento Grid 布局的视频集锦。

### 模块四：武印阁 (生态体系) - Ecosystem
- **六位一体导图**：交互式 SVG 能量流转图。
- **子公司详情**：响应式抽拉面板，展示六大实体公司职能。

### 模块五：合作入口 (B 端通道) - Cooperation
- **分流区块**：品牌、赛事、俱乐部、内容、投资五大申请通项。
- **统一申请表单**：“伟大的印章”风格表单，提交后触发盖章物理动画反馈。

## 4. 视觉规范 (Design Tokens)
- **Background**: `bg-[#050505]`
- **Primary**: `text-[#E61C23]` / `bg-[#E61C23]`
- **Accent**: `text-[#D4AF37]` (Imperial Gold)
- **Typography**: 
  - Display: 书法艺术字体
  - Sans: Source Han Sans
